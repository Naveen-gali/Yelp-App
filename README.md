# Yelp Api Implementation

A Basic Implementation of React Native and Yelp Api's.

### Tech Stack

| Library         | Category          |
| --------------- | ----------------- |
| React Native    | Mobile FrameWork  |
| React           | UI Framework      |
| MobX-State-Tree | State Management  |
| MobX-React-Lite | React Integration |
| TypeScript      | Language          |
